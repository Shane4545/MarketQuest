Scope (this build): Supported: Portfolio Allocation Optimizer (six modes), dollar-accurate splits summing to starting cash, full ranked_results table with recomputed multiples, targets & comparison. Not yet supported: arbitrary historical dates, multi-day daily switching, long-horizon path optimization. Data comes only from the imported JSON (ranked_results closes). One two-day window per file.

Portfolio Allocation Optimizer: Split starting cash across tickers any way that sums to your stake — e.g. $0.01 here, $10 there — using only imported entry/exit closes. Ending cash per line is allocated × (exit ÷ entry); portfolio ending is the sum.

Important: For a single two-day period, if there are no allocation constraints, perfect hindsight usually puts all cash into the highest-returning instrument. Splitting matters when you impose realism constraints such as maximum dollars per stock, maximum percent per stock, minimum number of holdings, or manual diversification.

Two-day perfect hindsight uses already-known closes. It is not a future prediction. Not financial advice. No brokers. No trades. No real-money orders.
