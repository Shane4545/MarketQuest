Governed Run Launcher

Start a fixture, dry-run, or live smoke acquisition run (and pipeline when acquisition is ready). This tool does not connect to brokers, send orders, or expose trade actions. Outputs are research and provenance artifacts only. See docs/FUTURE_TRADING_ARCHITECTURE.md for how real-money execution would be gated in a hypothetical future phase.

Open Run Viewer (read-only) — after a run completes, use ?run_id=… to jump to that run. Serve with python app/scripts/run_viewer_api.py so /api/* works.

Launcher merges safe trading defaults into JSON artifacts (no execution).
